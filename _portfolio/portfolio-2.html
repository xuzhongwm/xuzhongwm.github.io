--- title: "Java Wordle Game" excerpt: "A terminal-based clone of the popular Wordle game written in Java" collection: portfolio ---

This is a GUI-based clone of the popular word game Wordle, built using pure Java. The game randomly selects a five-letter word, and players have six attempts to guess it. After each guess, the program provides feedback on correct letters and their positions, mimicking the original Wordle gameplay mechanics.

  • GUI feedback with java ActionListener
  • Implement new modes such as Absurdle
  • Clean and modular object-oriented design
